FAQ

1. 连接失败
由于国内网络限制，连接失败主要有以下情况:
a. 连接不稳定，有时能连上有时不行。
  解决办法: 多尝试几次，一般能连上。或者试一下不同的VPN协议。
b. 手机可以，PC 不行。
  原因: 和其它软件冲突，比如360安全卫士。
  解决办法: 上VPN时退出360安全卫士或类似软件。
c. 有些网络可以，有些网络不行。比如中国移动网络可以，电信网络不行。 这是因为各个营运商对VPN的管制不同。
  解决办法: 尝试其他网络。或同a,尝试不同VPN协议。
d. 所有网络都连个不上。 原因是服务器被封锁。
  解决办法: 通知管理员转移服务器。

2. iPhone/iPad lose VPN connection in sleep (auto-lock) mode
It is an known issue on iOS devices. Some say Apple intend to drop the VPN connection in sleep mode when there is no traffic for privacy issues.
Android is working fine.

3. Error: Cannot find VPN server
Make sure you don't copy any white space into Server field. For example, ' us.petersvpn.com' will fail.

4. Windows 10 fails to connect L2TP/IPSec
You may need to change one registry setting and windows firewall settings. Please refer to below two articles.
Registry Settings: https://github.com/hwdsl2/setup-ipsec-vpn/blob/master/docs/clients.md#windows-error-809
Firewall Settings: https://www.magnumvpn.com/setup-windows-10-firewall-l2tp.html
